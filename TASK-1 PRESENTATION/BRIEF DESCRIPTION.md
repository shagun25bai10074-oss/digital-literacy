## BRIEF DESCRIPTION

I made a poster using the Canva tool that people can easily read and understand. This poster is about digital literacy which defines the importance of it. It talks about how to use the internet safely and what are the preventive measure we should take while using internet. It also talks about making a professional image online. It emphasizes the importance of practicing online safety by using strong passwords, protecting personal information, browsing securely, being aware of cyber threats, using security software, respecting others online, and logging out of shared devices. The poster shows how people can use skills to their advantage and how to keep their information safe. The digital skills poster is meant to help people understand this. I hope the poster will spread the message of using technology safely.
